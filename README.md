
Kendrick Lamar Fan Page

A fan-made website dedicated to the legendary rapper Kendrick Lamar. This site showcases his biography, albums, popular songs, and tour information. It’s a simple, interactive project built with HTML, CSS, and JavaScript — perfect for showcasing in your portfolio.

Features

✅ Biography section

✅ Albums list

✅ Music slider with navigation buttons

✅ Smooth scrolling navigation

✅ Responsive and modern design

✅ Footer with copyright

Demo

You can view the site locally by opening index.html in your browser.

Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and responsive layout

JavaScript – Music slider and smooth scrolling
